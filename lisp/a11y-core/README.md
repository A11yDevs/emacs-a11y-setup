# a11y-core

Pacote base com as fundações de acessibilidade compartilhadas pelo conjunto A11yDevs.

## Instalação

```elisp
(package-vc-install
 '(a11y-core
   :url "https://github.com/A11yDevs/emacs-a11y-setup.git"
   :branch "main"
   :lisp-dir "lisp/a11y-core"
   :main-file "a11y-core.el"))
```

## Uso

```elisp
(require 'a11y-core)
```

## Dependências

- Emacs 28.1+
