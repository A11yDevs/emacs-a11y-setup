# a11y-navigation

Pacote de navegação acessível com comandos e landmarks para o ambiente A11yDevs.

## Instalação

```elisp
(package-vc-install
 '(a11y-navigation
   :url "https://github.com/A11yDevs/emacs-a11y-setup.git"
   :branch "main"
   :lisp-dir "lisp/a11y-navigation"
   :main-file "a11y-navigation.el"))
```

## Uso

```elisp
(require 'a11y-navigation)
```

## Dependências

- Emacs 28.1+
- `a11y-core` 0.1+
