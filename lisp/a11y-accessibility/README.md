# a11y-accessibility

Pacote de funcionalidades de acessibilidade e suporte a leitor de tela para o ambiente A11yDevs.

## Instalação

```elisp
(package-vc-install
 '(a11y-accessibility
   :url "https://github.com/A11yDevs/emacs-a11y-setup.git"
   :branch "main"
   :lisp-dir "lisp/a11y-accessibility"
   :main-file "a11y-accessibility.el"))
```

## Uso

```elisp
(require 'a11y-accessibility)
```

## Dependências

- Emacs 28.1+
- `a11y-core` 0.1+
